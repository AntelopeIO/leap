## Description

The `faucet_testnet_plugin` provides an interface that assists in the automation of distributing tokens on an EOSIO testnet.

## Usage

```console
# config.ini
plugin = eosio::faucet_testnet_plugin
```
```sh
# command-line
nodeos ... --plugin eosio::faucet_testnet_plugin
```

## Options

None
